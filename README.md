# CVision

CVision adalah platform berbasis AI untuk menganalisis CV/Resume dan memberikan prediksi pekerjaan yang paling sesuai berdasarkan isi dokumen pengguna. Project ini merupakan kolaborasi antara tim Fullstack Web Development, Data Analyst, dan AI Engineer untuk membangun sistem rekrutmen modern yang lebih cepat, efisien, dan cerdas.

---

## 🚀 Features

- Upload CV/Resume (PDF/DOCX)
- Analisis isi resume menggunakan AI
- Prediksi pekerjaan yang sesuai
- ATS-friendly resume checking
- UI modern dan responsive
- Integrasi machine learning model
- Dashboard hasil analisis

## ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/NickyAditya/CapstoneProject-CVision.git
```

Masuk ke folder project:

```bash
cd CapstoneProject-CVision
```

Install dependencies:

```bash
npm install
```

Jalankan project:

```bash
npm run dev
```
